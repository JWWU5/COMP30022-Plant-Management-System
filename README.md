## Install

This project uses [node](http://nodejs.org) and [npm](https://npmjs.com). Go check them out if you don't have them locally installed.

Backend

```sh
$ cd pvsz-api
$ npm install

```

Frontend

```sh
$ cd pvsz-web
$ npm install
```

## Run

Backend

```sh
$ cd pvsz-api
$ npm run dev
```

Frontend

```sh
$ cd pvsz-web
$ npm run start
```
